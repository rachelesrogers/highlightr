# Transcript Example

Text corresponding to participant comments

## Usage

``` r
transcript_example
```

## Format

### `transcript_example`

A data frame with 1 row and 1 column:

- Text:

  Transcript text corresponding to the jury perception study

## Source

Jury Perception Study (see Rogers (2024)
<https://digitalcommons.unl.edu/dissertations/AAI31240449/> and Garrett
et. al. (2020) <doi:10.1037/lhb0000423>)
