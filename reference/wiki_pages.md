# Wikipedia Edit History for "Highlighter"

Text corresponding to versions of the Wikipedia article for Highlighter

## Usage

``` r
wiki_pages
```

## Format

### `wiki_pages`

A data frame with 300 rows and 1 column:

- page_notes:

  text of the Wikipedia page for Highlighter
