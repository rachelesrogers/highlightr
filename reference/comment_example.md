# Comment Example Dataset

Participant comments for the initial description used in the jury
perception study

## Usage

``` r
comment_example
```

## Format

### `comment_example`

A data frame with 125 rows and 2 columns:

- ID:

  Participant Identifier

- Notes:

  Participant notes

## Source

Jury Perception Study (see Rogers (2024)
<https://digitalcommons.unl.edu/dissertations/AAI31240449/>)
